SVBench documentation
=====================

:Author: Kez Cleal
:Version: 0.2.0

SVBench is a python module for parsing, benchmarking and classifying variant calls in .vcf or .csv format.

This module provides an API for manipulating variant files, and a command-line interface for applying
models to variant calls.

To install, type::

    pip install svbench

To satisfy requirements::

    pip install -r requirements.txt
